Moar-Net-Http
=============

cURL wrapper and utilities for making HTTP requests.

Part of the [Moar PHP Library][].

[![Build Status][ci-status]][ci-home]


Run the tests
-------------

    curl -sS https://getcomposer.org/installer | php
    php composer.phar install --dev
    phpunit
